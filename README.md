🧮 Smart Calculator (Java Swing)

A Smart Calculator desktop application built using Java Swing that performs basic arithmetic as well as advanced mathematical operations like trigonometric functions, logarithms, square root, and square.

This project demonstrates GUI programming, event handling, and mathematical computation in Java.

User-friendly Graphical User Interface (GUI)
Supports basic arithmetic operations:
Addition (+)
Subtraction (−)
Multiplication (×)
Division (÷)
Modulus (%)
Supports advanced operations:
Square root (√)
Square (x²)
Sine (sin)

📌 Features
Cosine (cos)
Tangent (tan)
Logarithm (log)
Backspace (←) functionality
Clear (C) button to reset calculator
Error handling for invalid inputs
Decimal precision formatting

🖥️ Technologies Used
Java
Java Swing
AWT
Event Handling
Math Library
No external libraries are required.

🚀 How to Run the Project
1️⃣ Prerequisites
Java JDK installed (version 8 or above)
Check installation:

java -version
2️⃣ Compile the Program
javac SmartCalculator.java
3️⃣ Run the Program
java SmartCalculator
🧪 How to Use
Enter numbers using the calculator buttons

Choose an operation (+, −, ×, ÷, %)

Press = to get the result

Use:

C → Clear all
← → Delete last digit
√ → Square root
x² → Square
sin, cos, tan → Trigonometric functions (in degrees)
log → Logarithm (base 10)
Result is displayed with proper decimal formatting
